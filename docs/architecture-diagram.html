AI Code Review Bot — System Architecture

End-to-end flow from GitHub webhooks through parallel Claude agents to inline PR feedback, persistence, and real-time dashboard updates.

Data shape: Each agent returns structured JSON with an issues[] array: file, line, severity, category, description, suggested_fix, fix_prompt. The aggregator merges, deduplicates, and ranks by severity before batch posting to the PR.
